# Coding-Sample

Most of my work is stored in private repositories due to agency data agreements and government projects. This repository provides representative code samples: a Python survey weighting module for the Maryland Commuter Survey, a Jupyter notebook demonstrating my approach to data assembly and feature engineering, and R code used for statistical analysis in a dissertation paper (two examples of data visualizations are included as well). These examples show how I use both Python and R for applied research, from data cleaning and standardization to modeling and inference.
